WORKOUT ENGINE — HIGH-LEVEL FLOW
REQUIRED FLOW (NON-NEGOTIABLE)

User opens Workout

App shows today’s workout preview

User taps Start Workout

Workout runs as a guided sequence with timers

Exercises + rest run automatically

Workout ends exactly at selected time

Completion is recorded

❌ No free skipping of exercises
❌ No jumping to end
❌ No auto-start without consent

WORKOUT STRUCTURE (MANDATORY)

Every workout MUST include:

1️⃣ Warm-up (15–20% of total time)

Joint mobility

Light cardio

Gradual ramp-up

Timed exercises

2️⃣ Main Workout (60–65% of total time)

Simple compound movements

Bodyweight or light resistance

No max effort

Time-based sets (not reps by default)

Example:

Squats – 45s

Push-ups (or wall push-ups) – 45s

Rest – 15s

Repeat

3️⃣ Cool-down (15–20% of total time)

Breathing

Stretching

Heart rate down-regulation

❌ A workout missing any section is INVALID.

TIMER-BASED EXECUTION (CRITICAL)
EXERCISE FLOW RULES

Each exercise has:

Name

Duration (seconds)

Instruction text

Timer starts ONLY when user taps Start

After each exercise:

Mandatory rest (default 10–20 seconds)

Rest duration may vary by:

experience level

fatigue signals

❌ No endless workouts
❌ No uncontrolled loops

USER CONTROL (ALLOWED)

User may:

Pause workout

Resume workout

End workout early (with confirmation)

If ended early:

Record as partial

Do NOT punish

Do NOT auto-extend later

NOTIFICATIONS & ENFORCEMENT (IMPORTANT)
DAILY WORKOUT REMINDERS

If workout not completed:

Send reminder notifications

Tone: firm but supportive

Example:

“You committed to 30 minutes today.
Let’s get it done — your body needs it.”

❌ No shaming
❌ No fear language

MISSED WORKOUT HANDLING

If workout not completed by end of day:

Mark as missed

Reflect in adherence

Do NOT force double workouts next day

Next day messaging:

“Yesterday didn’t go as planned.
Let’s show up today.”

fatigue reports

reduce exercise intensity

simplify movements

❌ System must NEVER increase time beyond user choice
❌ System must NEVER push harder because of missed days
